## Quick Reference

```aria
// Variable type
x: i32 = 42;

// Function parameter
fn process(data: string) { ... }

// Return type (uses ->)
fn calculate() -> i32 { ... }

// Struct field
struct User {
    name: string,
    age: i32
}
```

---

**Remember**: `:` declares **types**, `->` for **return types**!
