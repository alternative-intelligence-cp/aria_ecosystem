# The `flt128` Type

**Category**: Types → Floating Point  
**Syntax**: `flt128` (alias: `f128`)  
**Purpose**: 128-bit quadruple-precision floating point

---

## Overview

`flt128` provides **quadruple precision** for extreme accuracy requirements.

**Alias**: `f128`

---

## Specifications

| Aspect | Value |
|--------|-------|
| **Precision** | ~34 decimal digits |
| **Range** | ±1.2 × 10^4932 |
| **Storage** | 128 bits (16 bytes) |

---

## Declaration

```aria
// Type suffix required
precise: f128 = 3.14159265358979323846264338327950288f128;

// Alias
value: f128 = 1.0f128;
```

---

## Use Cases

### Scientific Computing

```aria
// Extreme precision calculations
avogadro: f128 = 6.02214076e23f128;
```

### Financial (Rare)

```aria
// Very high-value calculations
total: f128 = amount * interest_rate;
```

## Performance

```aria
// ⚠️ SLOW - software emulation
// Use only when f64 is insufficient
```

## Best Practices

### ✅ DO: Use for Extreme Precision

```aria
pi: f128 = 3.14159265358979323846f128;
```

### ❌ DON'T: Use Unless Necessary

```aria
// Overkill and slow
x: f128 = 1.0f128;  // ❌ Use f64
```
